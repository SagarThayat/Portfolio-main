## 🚀 Getting Started

### Option 1 — Direct Use
```bash
# Just open the file in any browser
open index.html
```

### Option 2 — Clone & Run
```bash
# Clone the repository
git clone https://github.com/SagarThayat/portfolio.git

# Navigate to folder
cd portfolio

# Open in browser
open index.html
# OR use Live Server in VS Code
```

> ✅ **No build step required** — pure HTML, CSS, JavaScript. Just open and run!

## 📬 Contact Form Setup

This portfolio uses **[Formspree](https://formspree.io)** for contact form:

1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form with your email
3. Replace the form endpoint in `index.html`:
```javascript
fetch('https://formspree.io/f/YOUR_FORM_ID', { ... })
```
